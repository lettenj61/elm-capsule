Aelm-capsule
============

A package that helps Elm go well with Bulma.


## Development

There are 2 ways to build and preview kitchen sink demo app.

### 1. Build it static

```sh
$ cd ./examples
$ yarn make
```

Then open `examples/index.html`.

### 2. Via `elm reactor`

```
$ cd ./examples
$ elm reactor
```

Then visit localhost.
